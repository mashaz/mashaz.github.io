shell-管道

一直在用” ps -ef | grep * “ 管理进程，上次被问到其中 ‘|’ 是干吗用的，我竟无言以对

那就来学习一下吧

管道命令与重定向区别

区别是：

1、左边的命令应该有标准输出 | 右边的命令应该接受标准输入
左边的命令应该有标准输出 > 右边只能是文件
左边的命令应该需要标准输入 < 右边只能是文件

2、管道触发两个子进程执行”|”两边的程序；而重定向是在一个进程内执行
